Student Management & Medical System
Overview

This project is a Student Management and Medical System designed to handle student registrations, medical records, application processing, and payment workflows. It provides a comprehensive dashboard experience for students, advisors, nurses, cashiers, and admins, allowing efficient management of applications and student data.
